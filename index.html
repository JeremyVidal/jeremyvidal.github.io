    I started my coding experience when attending college for my Associates in Computer Information Systems in 2012. Our focus was using PHP, MySQL along with HTML/CSS. After college I worked for an automotive dealership doing web Development. Where I was responsible for developing an application for the Sales department to assist in their daily duties.

    When I decided to get back into coding, I wanted something fresh and in-demand. The idea of college didn't seem the right path since it would take too long to get myself up and running. So, I looked into bootcamps. And the obvious choice was Eleven Fifty Academy.

    At Eleven Fifty Academy I learned extensive fundamentals of the Python language. With a heavy focus on Flask Web API’s using Postgres Database Management System. After some time, I felt I needed to add just one more skill to my arsenal. So back to Eleven Fifty I went and I signed up for their JavaScript class. This class focuses on a React front-end using Express for the back-end again using Postgres for the database management system.

Upon completion I am going to be ready to tackle all aspects of Web Development!
